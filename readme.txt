TLP Testimonial plugin is a fully Responsive to manage your company testimonial.


== Description ==

[Plugin Demo](http://demo.techlabpro.com/wp/tlptestimonial/) | [Documentation](http://demo.techlabpro.com/wp/tlptestimonial/documentation/)

TLP Testimonial plugin allow wordpress admin to add the testimonial with some very essential meta field like designation, location, company. It has also shortcode and widget area to display the output.

= Features =
* Fully Responsive
* Custom meta filed designation, location, company
* Shortcode
* Widget (carosule slider)

= Shortcode =
* [tlptestimonial number="4"]
* Here number is the total testimonial cout you want to display.

= Fully translatable =
* POT files included (/languages/)

= Available fields =
* Title/Name
* Description
* Designation
* Company
* Location

== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create testimonials.
4. Add shortcode or widget to display the testimonials.

= Requirements =
* **WordPress version:** >= 4
* **PHP version:** >= 5.2.4
